All data used in this study are publicly available. The Standardized Precipitation Evapotranspiration Index (SPEI) data were obtained from the official SPEI database (https://spei.csic.es/spei_database_2_11). The GPP datasets adopted in this research come from the GOSIF-GPP v2 product (https://data.globalecology.unh.edu/data/GOSIF-GPP_v2/). These two datasets can be accessed and downloaded free of charge. Considering the large file size of the original data, please visit the corresponding official websites to acquire the raw data as needed.
